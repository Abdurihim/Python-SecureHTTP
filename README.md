### 使用概述(Overview)

安装(Installation)

```bash
# 正式版(Release)
$ pip install -U SecureHTTP
# 开发版(Dev)
$ pip install -U git+https://github.com/staugur/Python-SecureHTTP.git
```

测试用例(TestCase)

```bash
$ make test
```
